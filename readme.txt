== Description ==

Provides easy access to "View Posts" for different post types in the admin bar and on the Dashboard in the admin area.

== Installation ==
1. If installing manually, unzip and copy the resulting directory to your plugins directory (default is : `wordpress_install/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
